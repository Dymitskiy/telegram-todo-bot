# Telegram ToDo Bot with Reminders
This is a Telegram ToDo bot that helps users manage tasks with categories, statuses, and reminders.
The bot is built with Python, Telebot, and Supabase as a backend.

## Run locally
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Create .env or environment variables:
   - BOT_TOKEN
   - SUPABASE_URL
   - SUPABASE_KEY
4. Run:
   python bot.py
